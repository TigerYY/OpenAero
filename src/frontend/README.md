# Frontend Components

这个目录包含所有前端组件和页面实现。

## 目录结构

```
frontend/
├── auth/           # 认证相关组件
├── solutions/      # 方案管理组件
├── admin/          # 管理员界面组件
├── shop/           # 商城组件
├── revenue/        # 收益管理组件
├── common/         # 通用组件
├── hooks/          # 自定义Hooks
├── contexts/       # React Context
└── types/          # 前端类型定义
```

## 技术栈

- React 18
- Next.js 14+
- TypeScript 5.0+
- Tailwind CSS
- Framer Motion
- Zustand (状态管理)
- React Hook Form
