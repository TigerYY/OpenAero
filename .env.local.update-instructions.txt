请将 .env.local 中的 DATABASE_URL 和 DIRECT_URL 更新为 Session Pooler 的连接字符串：

格式示例：
DATABASE_URL="postgresql://postgres.cardynuoazvaytvinxvm:[YOUR-PASSWORD]@aws-1-us-east-2.pooler.supabase.net:6543/postgres"
DIRECT_URL="postgresql://postgres.cardynuoazvaytvinxvm:[YOUR-PASSWORD]@aws-1-us-east-2.pooler.supabase.net:6543/postgres"

重要：
1. Host 必须是 .supabase.net（不是 .com）
2. Port 必须是 6543（不是 5432）
3. 替换 [YOUR-PASSWORD] 为你的实际数据库密码
4. 更新后需要重启开发服务器（npm run dev）
